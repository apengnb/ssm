凉山大火30条生命呼唤消防职业化

原创中国新闻网 2019-4-01 22:03:19
+10 +10 +10
一是消防员入伍时间短，灭火救援专业训练和实践经验缺乏，对灾害事故现场危及消防员安全的风险识别、评估、控制能力不足。二是消防员从事灭火救援时间短，灭火救援实践经验缺乏，灭火救援中紧急避险能力差。三是消防员入伍时间长，从事灾害事故组织指挥工作的可能性高，远离灾害事故危险区域，自身安全更有保障。 所谓“现役制”，即消防员主力是实行现役军人制的公安现役消防部队，约占全国消防队伍的6成。另外4成来自部分企业专职消防人员和义务消防组织。原公安现役消防部队属于武警序列，享受解放军同等待遇，归各级公安机关领导。消防队伍自省级（消防总队）、地市级（消防支队）到区、县（消防中队）依次而设。 一是消防员入伍时间短，灭火救援专业训练和实践经验缺乏，对灾害事故现场危及消防员安全的风险识别、评估、控制能力不足。二是消防员从事灭火救援时间短，灭火救援实践经验缺乏，灭火救援中紧急避险能力差。三是消防员入伍时间长，从事灾害事故组织指挥工作的可能性高，远离灾害事故危险区域，自身安全更有保障。 所谓“现役制”，即消防员主力是实行现役军人制的公安现役消防部队，约占全国消防队伍的6成。另外4成来自部分企业专职消防人员和义务消防组织。原公安现役消防部队属于武警序列，享受解放军同等待遇，归各级公安机关领导。消防队伍自省级（消防总队）、地市级（消防支队）到区、县（消防中队）依次而设。
